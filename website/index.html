StoreShot GitHub

App Store screenshots, without the design silo.

StoreShot is an open source visual editor and CLI that keeps your screenshot workflow open, local, and ready for humans, scripts, and AI agents.

Get started View on GitHub

From zero to canvas in a minute.

Install the CLI, create a project, and open the local visual editor. No account, database, or hosted backend required.

Terminal 
$ npm install --global storeshot
$ storeshot init app-store-screenshots \
    --app-name "Example App" --platform ios
$ storeshot dev app-store-screenshots
